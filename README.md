# Safa Hotel Network

Website B2B resmi Safa Hotel Network — menghubungkan agen travel umroh Indonesia dengan hotel-hotel bintang pilihan di Makkah dan Madinah.

## Tentang

Safa Hotel Network adalah mitra hotel resmi untuk agen umroh. Website ini ditujukan untuk agen perjalanan (B2B) yang membutuhkan akses ke harga khusus, konfirmasi instan, dan tim dedicated setiap musim.

## Fitur

- Animasi Ka'bah 3D interaktif di hero section
- Bilingual (Bahasa Indonesia / English)
- Informasi hotel di Makkah dan Madinah dengan detail fasilitas
- Formulir kontak via WhatsApp
- Modal detail hotel
- Syarat & Ketentuan dan Kebijakan Privasi
- Fully responsive (mobile, tablet, desktop)

## Teknologi

- HTML, CSS, JavaScript murni (single-file)
- Canvas API untuk animasi Ka'bah 3D
- Google Fonts: Cormorant Garamond, DM Sans
- Tidak ada framework atau dependency eksternal

## Struktur

```
safa-group/
└── index.html   # seluruh kode dalam satu file
```

## Pengembangan

Buka `index.html` langsung di browser — tidak perlu build step atau server.
